# firebase-node-react-demo

### Technologies stack
* **Node.js**
* **Firebase**
* **React.js**
* **Google API**
* **Facebook API**
* **Twitter API**


### Design/Front-end
* Material design

### Quick Start
##### 0. Prerequisites
* [Node.js is installed](https://nodejs.org/en/)
* [Npm dependencies is installed]
  
  - `npm install`

##### 1. Start the Node backend server
`node ./bin/www`
